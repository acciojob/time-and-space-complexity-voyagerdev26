Time and Space Complexity

Time complexity: How an algorithm's execution time increases with input size (N). How it's measured: By counting basic operations (comparisons, assignments). Hence it helps in determing efficient algorithm at a situation

Space Complexity: How an algorithm's memory usage scales with input size (N). How it's measured: Total memory (input + auxiliary space for variables, recursion, etc.).
